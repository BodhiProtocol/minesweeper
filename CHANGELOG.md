# Changelog

All notable changes to Minesweeper will be documented here.

## [1.0.0] — 2026-06-24

### Added
- Initial release of Minesweeper game
- Kids mode: 8×8 grid, 8 mines, 44px cells, fun color numbers
- Adult mode: 16×16 grid, 40 mines, classic number colors
- First-click safety guarantee (3×3 safe zone)
- Flood fill for zero-adjacent cells
- Right-click / long-press to flag
- Live HUD: mine counter, timer, smiley reset button
- Auto-flag remaining mines on win
- Dark mode via `prefers-color-scheme`
- SVG social preview image (1200×630)
- GitHub Pages deployment
